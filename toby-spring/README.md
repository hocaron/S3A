### 진행 방식
ver.1
1. 전 주에 발표할 사람을 지정한다.
2. 읽는다.
3. 발표하는 사람은 노션에 정리(깃헙에 올려도 됨)
4. 발표 순서 : 종윤 -> 선우 -> 세원 -> 종현 -> 도현

ver.2
1. 읽는다.
2. 다 같이 정리한다.
3. 매주 스터디 진행자는 랜덤으로 선정한다. (연속 X)
4. 스터디 진행 후 레포지토리에 업로드한다. (스터디 진행자가 담당)

### Our Ground Rules
